> 🔄 *Auto-updated daily via [GitHub Actions + lowlighter/metrics](https://github.com/lowlighter/metrics)*

<p align="center">
  <img src="github/metrics.plugin.isocalendar.fullyear.svg" alt="Isometric Commit Calendar" />

> ⚙️ **Setup:** Add the workflow below to `.github/workflows/metrics.yml` in your profile repo to activate this card.

<details>
<summary><b>📋 Click to see the GitHub Actions workflow for the Isometric Calendar</b></summary>

```yaml
name: Metrics
on:
  schedule:
    - cron: "0 0 * * *"   # runs every day at midnight UTC
  workflow_dispatch:

jobs:
  github-metrics:
    runs-on: ubuntu-latest
    permissions:
      contents: write
    steps:
      - uses: lowlighter/metrics@latest
        with:
          token: ${{ secrets.METRICS_TOKEN }}
          user: cochengannan
          template: classic
          base: ""
          plugin_isocalendar: yes
          plugin_isocalendar_duration: full-year
          filename: github/metrics.plugin.isocalendar.fullyear.svg
```

> Then create a GitHub Personal Access Token (PAT) with `public_repo` scope and add it as a secret named `METRICS_TOKEN` in your profile repo settings.

</details>

<p align="center">
  <img src="github/metrics.plugin.wakatime.svg" alt="WakaTime Stats" />
</p>

<details>
<summary><b>📋 Click to see the GitHub Actions workflow for WakaTime</b></summary>

```yaml
name: WakaTime Metrics
on:
  schedule:
    - cron: "0 0 * * 1"   # every Monday at midnight UTC
  workflow_dispatch:

jobs:
  wakatime-metrics:
    runs-on: ubuntu-latest
    permissions:
      contents: write
    steps:
      - uses: lowlighter/metrics@latest
        with:
          token: ${{ secrets.METRICS_TOKEN }}
          user: cochengannan
          template: classic
          base: ""
          plugin_wakatime: yes
          plugin_wakatime_token: ${{ secrets.WAKATIME_TOKEN }}
          plugin_wakatime_days: 7
          plugin_wakatime_sections: time, projects, projects-graphs, languages, languages-graphs, editors, os
          plugin_wakatime_limit: 5
          filename: github/metrics.plugin.wakatime.svg
```

> Add your WakaTime API key as `WAKATIME_TOKEN` secret. Get it from [wakatime.com/settings/api-key](https://wakatime.com/settings/api-key).
> Sign up free at [wakatime.com](https://wakatime.com) and install the IDE plugin to start tracking.
